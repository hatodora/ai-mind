# AIマインドマップアプリ 要件定義書

## コンセプト

> **「人間の脳で考えよう。行き詰まったらAIと対話して想像を膨らませよう」**

自分の頭でアイデアを出すことを主軸に置き、AIはあくまで「壁打ち相手」として機能するマインドマップWebアプリ。

---

## 1. 市場分析（既存ツールの問題点）

| ツール | 特徴 | 問題点 |
|--------|------|--------|
| Apple フリーボード | 無料 | Mac/iPad限定・スマホ操作が困難・Apple製品のみ |
| Gemini Mind Master | 画像解析→マップ変換・チャットAI | AIが全部やってしまう |
| Canva Mind Master | テキストデータの構造化特化 | ユーザー提供データ前提・創造支援が弱い |

### 既存アプリ共通の課題
1. **AIが全部やってしまう** → 自分の脳からアウトプットする機会が失われる
2. **自力で書くのも難しい** → 言葉が思い浮かばず進まない
3. **AI回答が難解** → 意識高い系コンサルタント風で理解しづらい
4. **共有のハードルが高い** → 自分の生の思考をそのまま人に見せるのは恥ずかしい

## 2. プロダクト要件

### 2-1. ターゲットユーザー
- アイデア整理・問題解決を必要とするすべての人（年齢問わず）
- スマートフォンを日常的に使うユーザー
- 「考えたいけど言語化が苦手」な人

### 2-2. コアコンセプト（差別化ポイント）
- **ヒューマンファースト**: AIに全部任せない。ユーザーが主役
- **ターン制協働**: ユーザーとAIが1ターンずつ交互に入力
- **どこでも使える**: スマートフォン前提のWebアプリ（電車・お風呂でも）
- **楽しさ**: ゲーミフィケーション要素で行き詰まりを防ぐ

### 2-3. 機能要件

#### 必須機能（MVP）
| # | 機能 | 説明 |
|---|------|------|
| F-01 | テーマ設定 | ユーザーが中心テーマを1つ入力してマインドマップ開始 |
| F-02 | ノード追加（手動） | ユーザーが自分でノードを追加 |
| F-03 | AIターン入力 | ユーザーのターンの後、AIが関連ノード候補を提案 |
| F-04 | ターン制UI | ユーザー→AI→ユーザー→AI の交互進行制御 |
| F-05 | マインドマップ表示 | ノードとエッジをビジュアルに表示・操作 |
| F-06 | 保存・読み込み | 作成したマインドマップをローカルまたはクラウドに保存 |

#### 重要機能（Phase 2）
| # | 機能 | 説明 |
|---|------|------|
| F-07 | 「わからない」ボタン | 選択ノードについてAIがわかりやすく説明 |
| F-08 | AI問題解決サポート | 完成マインドマップをAIがレビューし、解決策を提示 |
| F-09 | ユーザー認証 | アカウント登録・ログイン・マップ履歴管理 |
| F-10 | AIパーソナライズ | ターンを重ねることでAIがユーザーの思考スタイルを学習 |

#### 将来機能（Phase 3）
| # | 機能 | 説明 |
|---|------|------|
| F-11 | 他者意見取り込み | 他ユーザーの視点を匿名で取り込む機能 |
| F-12 | 共有機能 | マインドマップをURL共有・エクスポート |
| F-13 | ネイティブアプリ | iOS/Android ネイティブアプリ化 |
| F-14 | ゲーミフィケーション | 達成バッジ・連続使用ストリーク等 |

### 2-4. 非機能要件
- **レスポンシブデザイン**: スマートフォン（375px〜）を最優先
- **パフォーマンス**: AI応答 3秒以内
- **オフライン対応**: 基本操作はオフラインでも可能（Phase 2）
- **アクセシビリティ**: 子供から高齢者まで直感的に使えるUI

## 3. ユーザーフロー（MVP）

```
[アプリ起動]
    ↓
[テーマ入力] 例：「転職について考えたい」
    ↓
[マインドマップ画面]
    ↓
[ユーザーターン] ← 自分の考えをノードとして追加
    ↓
[AIターン] ← AIが関連するアイデアを2〜3個提案
    ↓
[ユーザーが選択・却下・追加]
    ↓
[繰り返し]
    ↓
[完成] → [AIによる全体レビュー・問題解決サポート]
    ↓
[保存 / 共有]
```

## 4. 画面構成

| 画面名 | 説明 |
|--------|------|
| ホーム | アプリ説明・新規マップ作成・過去マップ一覧 |
| テーマ設定 | 中心テーマ入力画面 |
| マインドマップエディタ | メイン画面（ノード操作・ターン制UI） |
| AI提案パネル | AIのターンで候補を表示するサイドパネル/ボトムシート |
| レビュー画面 | AIによる全体フィードバック |
| 設定 | プロフィール・AIの応答スタイル設定 |
